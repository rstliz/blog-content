# 変数の宣言
## 単数
```go
var firstName string

```

## 複数一括
```go
var (
    firstName, lastName string
    age int
)
```

## 初期化
```go
var (
    firstName string = "John"
    lastName  = "Doe" // 型推定
    age       int    = 32
)
```

# 定数
```go
const (
    StatusOK              = 0
    StatusConnectionReset = 1
    StatusOtherError      = 2
)```

# 注意点
変数も定数も宣言して、使用しないとエラーとなる
